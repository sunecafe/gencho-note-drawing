# Gencho Note 図面モード v0.1

明日の現調テスト用プロトタイプです。

## 入っている機能

- 部屋テンプレート追加
- 四角部屋の寸法入力
- 壁ごとのドア・窓・クローゼット・開口配置
- フリーハンド自由記入スペース
- 写真登録（カメラ起動 / 写真選択）
- 写真ごとの保存リンク
- 部屋メモ
- localStorage自動保存
- JSON出力

## 使い方

```bash
npm install
npm run dev
```

Vercelへそのままアップロードできます。

## 注意

Webアプリ版では、iOSの写真アプリへ完全自動保存は制限があります。
このv0.1では「撮影→アプリ登録→各写真の保存」まで対応しています。
ネイティブアプリ化した場合は、撮影と同時に写真アプリへ保存できます。
